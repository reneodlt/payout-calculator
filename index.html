How to Use

Simply click the "Go to Payout Calculator" button above to navigate to the calculator page. Once there:

  • Enter the tournament details such as entry fee, number of entries, rebuys, and add-ons.
  • Adjust the house fee type (percentage or absolute) and set the relevant values.
  • Define the payout structure and model parameters for accurate calculations.
  • View the results in the payout table and interactive chart.
